Colorful Select ============== Colorful Select is a free, open-source HTML modification plugin. Designed using primarily Javascript and jQuery.

Version 1.0

Colorful Select features:

  • Modify your dropdown menu to feature different shades of a specific color
  • Change the text color of the dropdown options
  • Choose to the invert the text color based on the background color
  • Choose from Dark text colors on Light Background and Light text colors on Dark Background
  • Change the font size and width of your select box directly from the plugin
